# FE setting with Webpack / babel / eslint / prettier

**기존에 구현한 가계부 프로젝트를 웹팩, 바벨 등을 이용하여 환경을 구성하여 구현합니다.**

- FE 프레임워크 - React
- BE 프레임워크 - Express
- DB - MySQL
- 개발환경 - Webpack + babel + eslint + prettier
